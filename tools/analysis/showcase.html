1. Star Intersection Map

Map all diagonal intersections of a {n/k} star polygon

What It Reveals

  • All intersection points colored by radius group
  • Concentric radius circles at each intersection distance
  • Parallel diagonal pairs highlighted in the sidebar
  • Hover tooltips showing which diagonals create each intersection
Key Insight for {10/4}:

40 total intersections lie on exactly 4 concentric radii. 5 parallel pairs exist (diag i and diag i+5), meaning these pairs create NO intersections in the interstitial zone.

Parameters

ParamDefaultDescription
n10Number of star tips
k4Star polygon skip
R0.55Star outer radius

2. Sector Decomposition

Analyze which diagonals pass through each interstitial gap

What It Reveals

  • Which 2 diagonals enter each sector gap
  • Whether those diagonals are parallel or intersecting
  • Separation distance between parallel pairs
  • Band edge lines (L/R offsets from center-line)
Key Insight for {10/4}:

ALL 10 sectors have parallel diagonal pairs (separation ~0.34). Face decomposition cannot find interstitial tiles here — must use per-sector explicit tile construction (Strategy 9) instead.

Parameters

ParamDefaultDescription
bandWidth0.025Half-width of strapwork bands
satDist0.64Satellite center distance
satR0.22Satellite radius
sector0Comma-separated sector indices

3. Tile Decomposition

Compute and label all tile vertices in one fundamental sector

What It Reveals

  • All tile shapes with labeled outlines (kite, rhombus, petal, wing, between-band)
  • Exact vertex coordinates with construction formulas
  • Toggleable layers: star tiles, interstitial, band edges, vertices
  • Edge lengths and vertex angles for each tile
Key Insight for {10/4}:

8 distinct tiles per sector: Kite, Rhombus A/B (star zone), Petal A/B, Between-band, Wing A/B (interstitial zone). Star tiles use diagonal intersection points; interstitial tiles use line-circle intersections.

Parameters

ParamDefaultDescription
sector0Which sector to analyze (0 to n-1)

4. Construction Overlay

Master visualization with all layers combined and toggleable

What It Reveals

  • Full pattern rendering with all geometric elements
  • 10 toggleable layers: diagonals, band edges, intersections, radii, sectors, star tiles, interstitial, satellites, strapwork, labels
  • Interactive parameter sliders for real-time adjustment
  • Statistics panel: tile counts, intersection count, radii
Key Insight:

Toggle layers on/off to see how construction lines create the tile pattern. Adjust sliders to explore how parameter changes affect the geometry in real time.
